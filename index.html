How this tool works

From real trail to Etsy-ready product · version 9.7

Version check: the version number also shows in the header, top-left. If it doesn't say v9.9 (or higher), you're running an old copy — delete other copies of this file and use only the latest. Old versions may put stale text on posters.

⚡ Quick Start — the whole job in 4 clicks

  1. Pick a trail from the dropdown (or City Map tab → type a place → Draw City Map)
  2. Click a style swatch you like
  3. 🗺 Save Route as GPX — do this once per trail, keep the file in a "WGS Trails" folder. Next time, drag that file in instead of searching: instant, works even when map servers are busy.
  4. 📦 Download Full Etsy Bundle — that ZIP is your finished product

If a search fails

"server busy / 504 / timeout" → the free public map server is overloaded. Wait 10–30 minutes and try again — or use a GPX you saved earlier, which never depends on the server.

"Failed to fetch" instantly → you're probably opening the file from your computer (file:///). This tool must run from its web address (your Netlify/hosted URL) — open the bookmark instead.

That's everything. No other troubleshooting exists for normal use.

Power features

  • ⚡ Batch mode (left panel): list several trails as "Name | Region" lines and get one master ZIP with every size for every trail, rendered in your current style.
  • ⭐ My defaults: save your preferred style/decorations/sizes once — every session starts pre-configured. Recent trails are listed there too for one-click reload.
  • 🔍 Verify route: overlays your drawn route on the real OpenStreetMap so you can eyeball accuracy before exporting. Preview only — never appears on exports.
  • 🖼 Wall mockup: one-click framed-on-a-wall PNG for a quick first listing photo.
  • Style variants (Export section): tick extra styles to export every colourway in one bundle — sell them as variants in a single listing.
  • Design extras: curved titles, three font pairings, four frame styles, two compass styles, and label position nudge sliders.
  • 🌊 Terrain edition (Design Options): the premium poster style — real lochs/lakes/rivers as filled shapes behind the route, named peaks and villages as waypoint labels, and a stats-grid footer (Distance · Highest Point · Total Ascent · Typical Duration). Click Fetch terrain & waypoints once per trail; everything is saved with the project. Notes: highest point & ascent need a GPX with elevation data; typical duration comes from Look Up Trail Facts; the sea is reconstructed from OSM coastline data; batch mode skips terrain.

✅ QA Gates — Wild Grid Studio standard

Layer 1 — Tool QA (run once whenever this tool gets a new version, before production use):

  1. Gate 1 · Structural: open the tool fresh — no errors, all buttons present. Load a project, save it, reload it: every setting must round-trip identically.
  2. Gate 2 · Functional: full workflow with the West Highland Way GPX as the known-good test fixture (correct answers: ~94.5 mi measured, coordinates 55.94°N 5.11°W → 56.82°N 4.32°W, Milngavie at the south/START). Load → style → facts → bundle export → open the ZIP and confirm every ticked size exists and renders → 🔍 Verify overlay → batch run with 2 trails → mockup → Word doc opens in Word.
  3. Gate 3 · Visual: your manual spot-check of exported PNGs at full zoom — typography, spacing, nothing overlapping.

Layer 2 — Product QA (every poster, before it is listed anywhere):

  1. 🔍 Verify route against the real map — the drawn line must follow the actual trail sensibly.
  2. Facts check: legend distance and established year confirmed against one official source (trail authority or Wikipedia).
  3. Start / Finish: geographically correct ends, labels not overlapping the route (use the nudge sliders).
  4. Export check: open the exported PNGs themselves — not just the on-screen preview — at full zoom.
  5. Test print one size at home for the first few products only, then spot-check occasionally.

Never skip or combine gates. A poster that fails any check does not get listed until fixed.

Step 1 — Pick a trail

Choose from the popular trails dropdown (pre-configured, one click) or type any trail/park name — suggestions appear as you type. The tool downloads that trail's real GPS route from OpenStreetMap. Every bend you see is the actual trail.

If a trail isn't found, download a free GPX file from AllTrails.com or Wikiloc.com and drop it in the upload box — that works for any trail on Earth, and also unlocks the elevation profile chart.

Design options (left panel)

  • Layout & line: portrait/landscape/square, line thickness, solid/dashed/dotted route, smoothing for very detailed routes.
  • Decorations: compass rose (ornate 8-point by default), corner coordinates, start/end markers with custom labels, background grid, frame, elevation profile, and a route outline so the line stays readable over terrain.
  • 🌊 Terrain edition: real water, woodland, moorland, glaciers, rivers and streams from live OSM data — plus the sea, reconstructed from coastline data so coastal routes no longer render the ocean as dry land. Water can be solid, hatched, or ruled lines.
  • 📍 Custom markers: click "Click on the map to place a marker", then click any spot on the poster. Click near a named peak or village and its name fills in automatically. Choose an icon first (pin, tent, peak, heart, star, dot). You can also type them: Label | place or lat,lon | icon. Labels auto-place to a free side and suppress any duplicate peak label beneath them. If a marker sits outside the picture, turn on Fit all markers inside the frame.
  • 🔲 Map framing: padding slider crops tight or leaves a wide margin; the shift sliders move the view when you're zoomed in.
  • Segment colours (Day-stage route segments): the alternating dashed/faded options now also include an alternating route-colour/accent-colour option — a coloured, block-by-block infographic look for multi-day trails.
  • Night sky accent: a seeded scatter of tiny flat stars across the upper third — only appears on dark styles (Dark, Midnight, Blueprint, Noir, Forest); does nothing on light styles.
  • Floating panel look (Terrain edition): pulls the map inward from the frame with a void margin and a thin rule, so it reads as an isolated inset panel rather than an edge-to-edge fill. Terrain mode only.
  • Keepsake log (bottom panel layout): turns the poster into a fill-in-after-your-hike keepsake — write-in lines for Date, Weather, Companions, Notes and Memory. Flat vector rules, no boxes — matches the map's own typography. Off by default; select it from the Bottom panel layout dropdown in Terrain edition.
  • 📊 Stats grid — manual override: type a Distance / Highest Point / Ascent / Duration and it beats the automatic value. Use it when a GPX has no elevation data. Whatever you type is printed exactly — check it against a real source.
  • Typography size: title and subtitle scale sliders, for very long or very short trail names.
  • Licensing: the OpenStreetMap credit line is required on anything you sell (ODbL). Keep it on Full wording for listings; position it with the X/Y sliders. Turning it off is possible but not licence-compliant.
  • Search: "Find Real Route" searches your region, then worldwide routes, then worldwide paths, and offers a pick-list when several trails match. Route segments are automatically stitched into true walking order, so distances are accurate. If search fails, use Test map-data connection to see which servers are blocked.
  • 📍 Custom markers (new in v9.9): add your own named points — a summit, a campsite, "where we met". Type one per line as Label | place or lat,lon. Place names are looked up automatically; coordinates plot instantly. Great for personalised gift posters. Saved with the project.
  • 🔲 Map framing (new in v9.9): the padding/zoom slider controls how tightly the poster crops the route — Tight crop fills the poster with the route, Wide margin shows more surrounding area. The two shift sliders move the view to reveal a different part of the surroundings when zoomed in. Reset by setting padding to Balanced and both shifts to 0.
  • Two new monochrome styles (new in v9.9): Noir (white on black) and Ink on White (black on white) — strict high-contrast looks that sell well as minimalist decor.
  • Custom colours: override any style's palette entirely — background, line, text, accent, frame.

Step 2 — Style it

Nine style swatches: Vintage, Minimal, Bold, Dark, Pastel, Blueprint, Forest, Desert, and Mono. Titles and subtitles have built-in suggestions; the legend has one-click templates.

Step 3 — Export

Quick PNG gives a single preview image. SVG is the editable master for Recraft/Illustrator. 📦 Full Etsy Bundle is the real product: tick the print sizes you want to sell, and it generates every one at true 300 DPI, plus the SVG and a ready-written "How to Print" guide for buyers, zipped into one file — upload that ZIP straight to your Etsy digital listing.

Print sizes explained

Standard Etsy poster bundles cover three ratio families so buyers can print any common frame size: 2:3 (covers 4×6" up to 24×36"), 3:4 (9×12, 18×24), 4:5 (8×10, 16×20), 11×14, and ISO A-sizes (A4–A1) for international buyers. Tick them all for maximum listing appeal — generation takes a few seconds per size.

The web-search buttons

Look Up Trail Facts finds real distance/elevation/established data for the legend. Research What's Selling reports current popular locations, styles, and pricing on Etsy. Both work automatically inside a Claude chat, or standalone with your API key (uses your API credits — roughly a cent or two per lookup). Always verify facts against an official source before publishing.

What this tool never does

It never invents geography. Routes always come from real map data or real GPS recordings. If neither source has the trail, it says so rather than guessing.

After export — remaining steps to a live listing

  1. Optional polish: open the SVG in Recraft/Illustrator for texture or extra typography.
  2. Visually check the route against a reference map (long trails can have small gaps between mapped segments).
  3. Test-print one size at home before listing.
  4. Create listing mockups (framed wall shots) with your existing Listing Asset Factory / Magnific workflow.
  5. Write the listing (title, tags, description) and check Etsy's Creativity Standards for the right creation-method disclosure.
